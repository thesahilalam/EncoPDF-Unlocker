**EncoPDF-Unlocker** is a high-performance, multi-threaded professional utility designed to recover lost passwords of PDF documents. Built with speed and efficiency in mind, it leverages 80% of your system's CPU power to crack passwords in record time.

## 🚀 Key Features
- **Multiprocessing Engine:** Uses 80% of available CPU cores for maximum speed.
- **Session Resume:** Automatically saves progress; resume anytime if interrupted.
- **Smart Modes:** 
  - **Brute Force:** Custom length (1-10) and charset selection.
  - **Wordlist Attack:** Integrated RockYou.txt downloader + Custom Wordlist support.
  - **Aadhaar/PAN Unlock:** Optimized patterns for Indian Government IDs.
- **Beautiful UI:** Modern terminal interface with real-time speed (Passwords/Sec).

## 💻 Manual Installation (For Developers)
If you have Python installed:
1. Clone the repo: `git clone https://github.com/thesahilalam/EncoPDF-Unlocker.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python src/main.py`
